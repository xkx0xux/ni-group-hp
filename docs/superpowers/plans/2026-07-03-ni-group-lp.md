# NIグループ トップページ（提案デモ）実装プラン

> **進め方メモ:** これは静的サイトの“見た目の作り込み”。各タスクの合格判定は「ブラウザ表示で崩れず・狙い通り・スマホでも綺麗」。各タスク末で必ずコミット。ヒーロー(Task 2)を最優先で作り、まずマスターに実物を見せる。

**ゴール:** NIグループの現行サイトを「明るい上質×紫の差し色×現代チック」に格上げした、トップ1枚長尺の提案デモを作り、GitHub Pagesに限定公開する。

**アーキテクチャ:** ビルドツールなしの静的サイト。`index.html` 1枚＋`assets/css/styles.css`＋`assets/js/main.js`。デザインはCSS変数（デザイントークン）で一元管理し全セクションで一貫。スクロール演出は IntersectionObserver。

**技術スタック:** HTML5 / CSS（カスタムプロパティ・Grid・Flex）/ 素のJavaScript。Webフォントは Google Fonts。ホスティングは GitHub Pages（noindex）。

## Global Constraints（全タスク共通・逐語で厳守）

**配色（ライト・白基調／紫は締めの差し色）**
- 背景: `--bg:#FFFFFF` / セクション替え `--bg-soft:#FAF9FC` / 紫の極淡面 `--bg-violet:#F5F3FF`
- カード面: `--surface:#FFFFFF` 枠 `--border:#ECEAF2` 影は淡く広く
- 文字: 主 `--ink:#17141F` / 副 `--ink-2:#5B5566` / 反転 `--ink-inv:#FFFFFF`
- アクセント紫: `--accent:#6D28D9` / 濃 `--accent-deep:#4C1D95` / 淡 `--accent-soft:#8B5CF6`
- グラデ（ヒーロー等の差し色）: `#7C3AED → #4F46E5`（低彩度・面積は小さく）
- ※紫は「見出しの一部・線・小さな図形・CTA」だけに使い、面で塗らない。基調はあくまで白。

**タイポグラフィ（Google Fonts）**
- 和文見出し: `Zen Kaku Gothic New`（700/900）
- 和文本文: `Noto Sans JP`（400/500）
- 欧文/数字ディスプレイ: `Sora`（600/700）
- 欧文本文: `Inter`（400/500）
- 見出しは大きく・字間やや詰め・行間ゆったり。数字カウンターは `Sora`。

**レイアウト/間隔**
- コンテナ最大幅 `1200px`、左右パディング 24px（スマホ）。
- セクション上下padding: PC 120px / スマホ 72px。8pxグリッドで余白を統一。

**モーション**
- 出現: フェード＋16px上げ、480〜600ms ease-out、IntersectionObserverで一度だけ。
- 数字: ビューポート入りでカウントアップ。
- `@media (prefers-reduced-motion: reduce)` で演出オフ（即表示）。
- ホバーは上品に（影・わずかな持ち上げ・下線の伸び）。過剰な動きは禁止。

**レスポンシブ/品質**
- PC・タブレット・スマホで崩れない。ブレークポイント目安 1024 / 768 / 480。
- 画像は最適化、遅延読み込み。外部JSライブラリ非依存。
- アクセシビリティ: 文字コントラストAA、`alt`、キーボード到達、`:focus-visible`。

**公開制約**
- `robots.txt` と `<meta name="robots" content="noindex,nofollow">` で検索よけ。
- ドメイン非依存（絶対URL埋め込み禁止・相対パス）。`.nojekyll` を置く。

**内容（実情報を使用・ロレム禁止）**
- 5社: ①株式会社New innovation＝空調・配管・電気の設備工事 ②株式会社JMC＝塗装 ③株式会社New bloom＝人材・BPO・SES・採用コンサル ④株式会社Sales Finder＝クラウド・システム開発・営業コンサル ⑤合同会社New space＝労務・助成金コンサル
- 理念: 「守れる人の数を増やしたい」
- CEO: 内田 雄真。メッセージ本文＝現行サイトの文面を流用（設計メモ参照）。
- ニュース3件: 2025-12-16「カスタマーハラスメントに対する当社の基本方針について」/ 2025-11-28「社員の井上 海選手が第8回日本社会人選手権水泳競技大会に出場＆入賞」/ 2025-06-16「登録電気工事の免許を取得しました」
- 数字（※要確認・仮置き）: グループ5社(確定) / 事業領域5 / 設立 仮2023 / 社員数 仮"20代中心の若い組織"を数字化する場合は要確認。実数字が来るまで断定的な数字は控えめに。

### Task 1: プロジェクト骨組み＋デザイントークン＋ヘッダー/フッターの器

**Files:** Create `index.html`, `assets/css/styles.css`, `assets/js/main.js`, `robots.txt`, `.nojekyll`

- HTMLの土台: `<head>`にmeta（charset/viewport/`noindex`/OGP最小/title「NI Group｜守れる人の数を増やす」）、Google Fontsリンク、`styles.css`読込。`<body>`にヘッダー・`<main>`空セクション骨組み・フッター・`main.js`。
- `styles.css`: 先頭に`:root`で上記トークン全定義→軽量リセット→コンテナ/セクション/見出しの共通クラス→ヘッダー/フッター。
- ヘッダー: 白・半透明・スクロールで薄い影が出るスティッキー。左にNIロゴ（当面テキストロゴ「NI GROUP」）、右にナビ（グループ概要/事業内容/ニュース/お問い合わせ）＋紫のCTAボタン。スマホはハンバーガー。
- フッター: 会社名「株式会社New innovation」・5社ナビ・連絡先（`innovation1017@new2023.jp`）・小さくロゴ。整った企業フッター。
- **合格判定:** ブラウザで開くと、白基調・上品なフォント・スティッキーヘッダー・フッターが表示され、スマホ幅でも崩れない。
- **Commit:** `feat: 骨組み・デザイントークン・ヘッダー/フッター`
