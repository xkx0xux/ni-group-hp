# NIグループHP（提案デモ）— このタブの運用ルール

## これは何
マスターの前弊社「NIグループ」公式サイト（https://www.ni-holdings.com ）の**見た目格上げ提案デモ**。
本番サイトは触らない。限定URLを内田雄真CEOらに見せて「Before→After」で提案するのが目的。

## 絶対ルール
- **本番（ni-holdings.com）には一切手を出さない**。あくまで別置きのデモ。
- 会社情報（5社・理念・CEO名・ニュース）は現行サイトの実情報を使う。**勝手な数字の捏造禁止**（社員数・設立年など未確認情報は載せない）。
- 検索よけ（noindex）を必ず維持。先方に無断でデモが検索に載る事故を防ぐ。
- デザインは「夜明け」構成（夜→夜明け→昼→夕）。白基調＋紫は差し色、の原則を崩さない。
- 派手にしすぎない。上品・シネマティック・2026年の今どき、が基準。

## 技術の決めごと
- 静的サイト（HTML/CSS/JS 各1ファイル）。ビルドツール・Reactは使わない。
- 動きは GSAP + ScrollTrigger + Lenis（CDN読み込み）。**未読込時・reduced-motion時は静的表示に自動フォールバック**する設計を維持。
- SVGの縦線に注意：グラデ・filterは userSpaceOnUse 指定必須（幅ゼロbbox問題）。
- 確認は必ず「ローカルサーバー経由」（file://はPlaywrightで開けない）：
  `python3 -m http.server 5510 --bind 127.0.0.1`（~/NIグループHP で）

## 作業の型
- 変更→Playwrightスクショ検品（PC 1440px／スマホ 390px）→コミット。
- 見た目の変更はマスターに実物URLで見せてから次へ。
- モデル：デザインの重い作業はFable5推奨（マスターに/model切替を提案）。
